How it works

Four steps.
That's it.

1

Install and launch

Run TrackIt.exe. It lives in your system tray — no installation wizard, no admin rights needed. Opens on Windows startup if you want.

2

Open a project in any supported app

TrackIt detects the foreground window. First time it sees a new project file, a small popup asks: New project detected — want to track it? Hit Yes once.

3

Work normally

The timer runs while your project is in focus. Switch to Lumetri, export dialogs, or other panels — it keeps running. Switch to Chrome or Discord — it pauses automatically.

4

Invoice from the History tab

Go to History → select sessions → Generate Invoice. Edit the HTML preview inside the app, then open in browser and print or export to PDF.
